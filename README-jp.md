## 開発動機

AnDropは、AndroidデバイスからmacOSデバイスにWi-Fi経由でファイルを転送するための、最もシンプルで最高のユーザー体験を提供することを目的としています。つまり、セットアップの必要がなく、発見が早く、転送速度が速いということです。

AnDropは、Android OSの共有メニューに統合されているため、最小限のUIで動作し、使用時の摩擦を減らすことができます。
